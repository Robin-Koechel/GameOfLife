# GameOfLife
"Conway's Game of Life" implementation in C++

## How to run:
1. git clone https://github.com/Robin-Koechel/GameOfLife.git
2. cd GameOfLife
3. mkdir build
4. cd build
5. cmake ..
6. make
7. ./gameOfLife <width> <height> <tickSpeedInSeconds>
