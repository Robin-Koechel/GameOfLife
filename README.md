# GameOfLife
"Conway's Game of Life" implementation in C++

## How to run:
````
git clone https://github.com/Robin-Koechel/GameOfLife.git
cd GameOfLife
mkdir build
cd build
cmake ..
make
./gameOfLife <width> <height> <tickSpeedInSeconds>
````
